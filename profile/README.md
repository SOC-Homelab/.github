# SOC-Homelab

Blue team home lab built from scratch: centralized SIEM, real attack simulation, detection analysis.

**[siem-wazuh-homelab](https://github.com/SOC-Homelab/siem-wazuh-homelab)** — Wazuh Manager and agent on Ubuntu Server, Kali attacker machines, three physical hosts on a bridged network. Confirmed detections: Nmap reconnaissance, Hydra SSH brute force, File Integrity Monitoring via auditd.

`Wazuh` `Ubuntu Server` `Kali Linux` `VirtualBox` `auditd` `Nmap` `Hydra` `Wireshark`
